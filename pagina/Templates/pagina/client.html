Bootstrap 4 Responsive Type Scale

OK, so we know Bootstrap does not come with a responsive type scale as such, but it does come with all you need to set one up. This is my way of getting things set up to help overcome a couple of dislikes I have when browsing through the web.

I have an issuse with RWD at the moment and that is the way some sites seem quite happy to fill a mobile screen with a page title or heading. Lets try and rectify that.

Take a quick read over the Responsive Typograpy section of the V4 docs. This will let you understand exactly what Bootstrap sets up for you and how so we know were to start.

The next thing to consider is some details I read over on CSS Tricks about the use of rems and ems. The only issue you will come across if you want to implement this kind strategey when using Bootstrap is that V4 sets all the typography font-size's with rem. If you want to use the suggestion from CSS Tricks you will need to either override the styles as done in this pen or edit the cource code.
